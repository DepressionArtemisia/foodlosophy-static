# Foodlosophy Static

Static responsive recreation of the visible Foodlosophy WordPress home page.

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL printed by Vite.

Default dev URL:

```text
http://127.0.0.1:5174/
```

## Build

```bash
npm run build
```

## Notes

- Legitimate Foodlosophy images, logo, favicon, and local Nunito fonts are stored in `public/assets/foodlosophy`.
- External links are limited to ChowNow, Instagram, telephone, email, and the visible footer credit.
- WordPress, Elementor scripts, tracking scripts, admin code, and hidden injected content were not copied.
